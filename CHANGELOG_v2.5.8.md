# TinyLedger v2.5.8 Release Notes

**发布日期**: 2026-04-21

## 🎉 新功能

### 统一音效和震动反馈系统
- ✨ **保存音效**: 记账保存成功时播放水滴声 (`waterdrop_notificationevening1.ogg`)
- 🗑️ **删除音效**: 删除账单时播放删除音效 (`system_delete.ogg`)
- 📳 **震动反馈**: 所有保存和删除操作均支持震动反馈（50ms）
- ⚙️ **智能控制**: 自动读取"我的"页面中的"声音"和"震动"开关设置

### 支出详情页优化
- 🖼️ **多图片支持**: 支持显示以 `||` 分隔的多张账单图片
- 📸 **图片缩略图**: 横向展示所有已拍摄的照片缩略图
- 🔍 **全屏预览**: 点击图片可全屏预览，多张图片支持左右滑动浏览
- 📱 **页码指示**: 多图预览时显示页码指示器（如 "2 / 3"）
- 🏷️ **账户名称**: 显示真实的账户名称，替代硬编码文本

### 账单相册页增强
- 🖼️ **图片显示**: 账单卡片显示第一张已拍摄的照片缩略图
- 📷 **照片图标**: 有图片的账单在账单列表中显示照片图标
- 📐 **尺寸适配**: 修复图标在不同字体缩放下的显示问题

## 🐛 修复

### TransactionDetail 页面修复
- 🔧 **数据加载**: 修复详情页因 ViewModel 数据为空导致无法打开的问题
- 💾 **独立查询**: 改为通过 Repository 直接按 ID 查询交易记录
- 🚀 **性能优化**: 不再依赖 BillsViewModel 的内存数据

### 多图片路径解析修复
- 🔧 **分隔符处理**: 正确处理 `imagePath` 字段中的 `||` 分隔符
- 📸 **Coil 加载**: 使用 `File(path)` 作为图片模型，避免 URI 解析失败
- 🖼️ **缩略图展示**: 相册页只显示第一张图片，避免性能问题

## 🛠️ 技术改进

### 架构优化
- 🏗️ **新建 ViewModel**: 创建 `TransactionDetailViewModel` 独立管理详情页数据
- 📦 **依赖注入**: 使用 Hilt 正确注入 Repository
- 🔄 **状态管理**: 统一使用 `StateFlow` 管理 UI 状态

### 音效管理
- 🎵 **SoundFeedbackManager**: 新建统一音效管理器
- 🔊 **MediaPlayer**: 使用 MediaPlayer 播放 OGG 音频文件
- 📳 **震动兼容**: 支持 Android 13+ 的 VibratorManager API
- ⚙️ **设置联动**: 实时读取用户的音效和震动偏好设置

### UI 组件修复
- 📐 **图标尺寸**: 使用 `LocalDensity` 正确转换 sp 到 dp
- ♿ **无障碍**: 修复字体放大时图标与文字大小不一致的问题
- 🎨 **Material 3**: 保持与 Material Design 3 规范一致
