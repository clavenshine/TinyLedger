# 小小记账本 - 项目规范

## 1. 项目概述

**项目名称**: 小小记账本 (TinyLedger)
**项目类型**: Android 原生应用
**核心功能**: 简洁易用的个人记账应用，支持收入/支出记录、账户管理、自动导入、分类管理、统计报表和数据导出
**目标用户**: 普通消费者，需要简单快捷的日常记账体验

## 2. 技术栈

| 类别 | 技术选型 | 版本 |
|------|----------|------|
| 语言 | Kotlin | 1.9.x |
| UI框架 | Jetpack Compose | 1.6.x (BOM 2024.02.00) |
| 架构 | MVVM + Clean Architecture | - |
| 状态管理 | StateFlow / Compose State | - |
| 本地数据库 | Room | 2.6.x |
| 依赖注入 | Hilt | 2.50 |
| 导航 | Navigation Compose | 2.7.x |
| 最低SDK | 26 (Android 8.0) | - |
| 目标SDK | 34 (Android 14) | - |
| 编译SDK | 35 | - |

## 3. 功能模块

### 3.1 记账功能
- ✅ 添加收入/支出记录
- ✅ 选择分类（支持新增自定义分类）
- ✅ 输入金额（支持小数）
- ✅ 添加备注说明
- ✅ 选择日期（默认当天）
- ✅ 编辑/删除已有记录
- ✅ 选择账户

### 3.2 账户管理
- ✅ 我的账户管理
- ✅ 支持账户类型：银行卡、微信、支付宝、现金、其他
- ✅ 期初余额初始化
- ✅ 账户余额自动更新（期初余额 + 收入 - 支出）
- ✅ 账户颜色自定义
- ✅ 账户详情展开/折叠（显示收支明细）
- ✅ 按月分组的交易明细（可展开查看具体日期）
- ✅ 账户卡片左滑显示编辑/删除选项

### 3.3 自动导入
- ✅ 短信读取权限
- ✅ 银行短信自动识别收入/支出
- ✅ 微信支付短信自动识别
- ✅ 支付宝短信自动识别
- ✅ 批量导入选择

### 3.4 账单管理
- ✅ 按月查看账单列表
- ✅ 按类型筛选（全部/收入/支出）
- ✅ 按分类筛选
- ✅ 搜索账单（按备注）

### 3.5 统计报表
- ✅ 月度收支概览（总收入、总支出、结余）
- ✅ 支出分类占比（饼图）
- ✅ 月度趋势（柱状图）

### 3.6 数据管理
- ✅ 数据持久化存储（Room数据库）
- ✅ 账户数据独立管理

### 3.7 设置
- ✅ 货币符号设置
- ✅ 主题切换（浅色/深色/跟随系统）
- ✅ 账户管理入口
- ✅ 自动导入入口

## 4. UI/UX 设计规范

### 4.1 设计风格
- **风格**: iOS风格 + Material Design 3
- **主题**: 清新简约，适合金融类应用
- **圆角**: 20dp（大卡片）、16dp（卡片）、12dp（按钮）
- **动画**: 丝滑流畅的页面切换动画

### 4.2 配色方案

**浅色主题 (iOS风格)**:
| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色 | iOS蓝 | #007AFF |
| 收入色 | 系统绿 | #34C759 |
| 支出色 | 系统红 | #FF3B30 |
| 背景 | 浅灰 | #F2F2F7 |
| 表面 | 白色 | #FFFFFF |

**深色主题**:
| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色 | iOS蓝 | #0A84FF |
| 收入色 | 系统绿 | #30D158 |
| 支出色 | 系统红 | #FF453A |
| 背景 | 纯黑 | #000000 |
| 表面 | 深灰 | #1C1C1E |

### 4.3 字体规范
- **标题**: SF Pro Display, 20-24sp, SemiBold
- **正文**: SF Pro Text, 14-16sp, Regular
- **金额数字**: SF Pro Rounded, 24-32sp, Bold

### 4.4 导航结构
```
底部导航栏 (4个Tab)
├── 记账 (首页/添加记录)
├── 账单 (列表/筛选)
├── 账户 (我的账户管理)
└── 统计 (报表/图表)
```

### 4.5 动画效果
- Tab切换: 300ms 滑动+淡入淡出
- 列表项: 交错进入动画
- 按钮点击: 弹性缩放效果
- 对话框: iOS风格圆角+模糊背景

## 5. 数据库设计

### 5.1 账目表 (transactions)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long (PK) | 自增ID |
| type | Int | 0=支出, 1=收入 |
| category | String | 分类键值 |
| amount | Double | 金额 |
| note | String? | 备注 |
| date | Long | 时间戳 |
| accountId | Long? | 关联账户ID |
| created_at | Long | 创建时间 |
| updated_at | Long | 更新时间 |

### 5.2 账户表 (accounts)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long (PK) | 自增ID |
| name | String | 账户名称 |
| type | String | bank/wechat/alipay/cash/other |
| icon | String | 图标名称 |
| initialBalance | Double | 期初余额 |
| currentBalance | Double | 当前余额 |
| color | String | 账户颜色 |
| cardNumber | String? | 卡号后4位 |
| isEnabled | Boolean | 是否启用 |

### 5.3 分类表 (categories)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (PK) | 分类键值 |
| name | String | 显示名称 |
| icon | String | 图标名称 |
| type | Int | 0=支出, 1=收入 |
| is_default | Boolean | 是否默认分类 |

### 5.4 预置分类

**支出分类**:
- 餐饮、交通、购物、娱乐、居住、医疗、教育、通讯、储蓄、运动、宠物、孩子、其他支出

**收入分类**:
- 工资、奖金、投资、兼职、红包、其他收入
