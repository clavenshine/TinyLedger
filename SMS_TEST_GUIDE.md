# TinyLedger 短信解析和通知功能测试指南

## 📋 测试前准备

### 1. 确认通知监听服务已启用

在小米设备上：
1. 打开 **设置** → **通知与控制中心** → **通知使用权**
2. 找到 **小小记账本通知监听**
3. 确保已**开启**

### 2. 确认应用设置已开启

打开 TinyLedger 应用：
1. 进入 **设置** → **自动记账**
2. 确保以下选项已开启：
   - ✅ **自动记账功能**
   - ✅ **银行短信捕获**
   - ✅ **提示音**（可选）
   - ✅ **震动**（可选）

## 🔍 日志检查方法

### 方法1：实时监控日志

在电脑上运行：
```bash
adb logcat | Select-String -Pattern "TxNotifService|SmsReceiver|SmsTransactionParser"
```

**关键日志标识：**
- `[通知捕获]` - 通知监听服务捕获到通知
- `捕获银行短信通知` - 识别到银行短信
- `银行短信已存储` - 短信已保存到数据库
- `银行短信无感自动记账成功` - 无感模式下记账成功
- `银行短信待确认记账已保存` - 待确认模式
- `支付通知 解析成功` - 微信/支付宝通知解析成功

### 方法2：查看历史日志

```bash
adb logcat -d | Select-String -Pattern "TxNotifService" | Select-Object -Last 50
```

## 🛠 故障排查

### 问题1：通知未被捕获

**检查项：**
1. ✅ 通知使用权是否已开启？
   - 设置 → 通知与控制中心 → 通知使用权 → 小小记账本通知监听
2. ✅ 自动记账功能是否已开启？
   - 打开 TinyLedger → 设置 → 自动记账
3. ✅ 银行短信捕获是否已开启？
   - 同上路径

**解决方法：**
- 关闭并重新开启通知使用权
- 重启 TinyLedger 应用
- 重启设备

---

### 问题2：短信解析失败

**检查日志：**
```bash
adb logcat | Select-String -Pattern "SmsTransactionParser"
```

**可能原因：**
1. 短信格式不符合已知银行模板
2. 金额提取失败
3. 银行来源识别失败

**解决方法：**
- 查看日志中的具体错误信息
- 确认短信包含关键词（如"建设银行"、"支出"、"元"等）
- 确认金额格式正确（如"100.00元"）

---

### 问题3：重复记账

**检查项：**
- 是否同时开启了短信广播接收器和通知监听？
- 正常情况下，通知监听优先，短信接收器会跳过

**解决方法：**
- 日志中应显示：`Notification listener active for bank SMS, skipping SmsReceiver auto-accounting to avoid duplicates`
- 如果仍然重复，检查通知监听服务是否正常运行

---

### 问题4：小米设备特殊问题

**小米 HyperOS/MIUI 注意事项：**
1. **通知使用权** ≠ **通知权限**
   - 通知使用权：设置 → 通知与控制中心 → 通知使用权
   - 通知权限：设置 → 应用管理 → TinyLedger → 权限管理 → 通知

2. **短信应用包名适配**
   - TinyLedger 已适配小米短信包名：`com.android.mms`、`com.miui.sms`、`com.xiaomi.sms`

3. **SIM卡短信查询**
   - 应用已支持查询 SIM 卡存储的短信
   - URI：`content://sms/icc`

4. **后台进程保护**
   - 将 TinyLedger 加入后台锁定列表
   - 设置 → 应用设置 → 应用锁 → 锁定 TinyLedger

## 📝 手动测试步骤（无需真实银行短信）

### 方法1：通过短信应用发送测试短信

1. 打开小米设备的**短信**应用
2. 创建新短信，收件人填写：`106980095533`
3. 内容填写：`【建设银行】您尾号1234的储蓄卡12月15日20时23分支出100.00元，可用余额5000.00元。`
4. **不要发送**，而是**保存为草稿**
5. 打开 TinyLedger，进入**短信导入**功能
6. 点击**扫描短信**，查看是否能识别该草稿短信

### 方法2：使用真实交易测试

1. **微信测试：**
   - 向自己的另一个微信账号转账 0.01 元
   - 观察是否弹出记账确认通知

2. **支付宝测试：**
   - 使用支付宝扫描商户二维码（小额）
   - 或向自己的另一个支付宝账号转账

3. **银行短信测试：**
   - 等待真实银行交易短信
   - 或请朋友向你转账并观察短信通知
